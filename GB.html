About Project - GrowingBamboo

In the summer of 2015, I proposed this game idea to my best friend (Feihong Ren - Programmer) and we decided to work on this 2D game project which named "GrowingBamboo". This game is a stacking game. The principle of the game is "Simple" (Simple to Understand), "Easy" (Easy to Play), "Addictive" (Addict to Win).

Please check out the design detail below:

1.

Home Page

This is game main page, user has options to view scores, rate the game, Ads, audio adjustment, and tutorial of how to play the game, etc. User simply tap the "tap to play" to start the game.

2.

Game Starts

After user tap to start the game, background animation will start, a single bamboo will appear and floating/bouncing from left to right and right to left repeatedly. User can tap anywhere on the screen to stop/stack it.

3.

Game Over

User will be prompted to this "game over" page when he/she lose the game. This page will show user total score. User can choose to go back to home page or play again. User also has options to rate the game or share the score to friends on socail media.
